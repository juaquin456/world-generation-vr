# world-generation-vr

Bienvenido a World Generation VR, un juego de exploración inmersivo desarrollado en Unity para dispositivos de realidad virtual. Inspirado en la aclamada mecánica de exploración de Outer Wilds, nuestro juego te lleva a través de diversos mundos generados proceduralmente, ofreciendo una experiencia única cada vez que juegas. Embárcate en una aventura para descubrir nuevos entornos, interactuar con objetos misteriosos y desentrañar los secretos del universo, todo en un entorno VR completamente envolvente.

Este proyecto es una demostración temprana de lo que será una experiencia rica y dinámica de exploración espacial. Únete a nosotros en este viaje a través de mundos desconocidos.
